# Permalist Website

A simple, lightweight website to create and manage permanent links to your favorite resources. Perfect for keeping important links organized in one place. One of my learning project to understand how postgresSQL works when implemented into the website for storing data.

## Usage

Clone the repository:  
   ```bash
   git clone https://github.com/yourusername/permalist-website.git
   ```
